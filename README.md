# 🌍 EcoSphere – ESG Management Platform

## Overview

EcoSphere is a centralized ESG (Environmental, Social, and Governance) management platform designed to help organizations monitor sustainability initiatives, employee engagement, and compliance activities.

The platform enables companies to track carbon emissions, manage CSR programs, monitor governance policies, and generate ESG reports through an intuitive dashboard.

---

# 🎯 Objective

Build a scalable ESG solution that allows organizations to:

- Measure environmental impact.
- Organize social responsibility activities.
- Track governance and compliance.
- Encourage employee participation through gamification.
- Generate reports and analytics.

# 📈 ESG Score Formula

```text
Total ESG Score =

(Environmental Score × 40%)

+ (Social Score × 30%)

+ (Governance Score × 30%)
```

# 🛠️ Tech Stack

Since this project does not use built-in Odoo modules, the implementation follows a custom architecture:

- Backend: Python
- Database: PostgreSQL
- API: REST API
- Frontend: HTML, CSS, JavaScript / React
- Charts: Chart.js
- Authentication: JWT
- Reports: PDF Generator
